
## Печать итоговых вопросов в PDF

### Что нужно сделать
Добавить кнопку "Скачать PDF" в модалку "Итоги менторства" (Точка Б), которая сформирует PDF-документ с вопросами и ответами клиента.

### Подход
Использовать встроенный браузерный API `window.print()` через специально сформированное скрытое окно. Это **не требует установки дополнительных библиотек** и работает во всех современных браузерах.

### Как это будет работать
1. Пользователь заполняет ответы на итоговые вопросы
2. Нажимает кнопку "Скачать PDF" внизу модалки
3. Открывается системное окно печати с отформатированным документом
4. Пользователь выбирает "Сохранить как PDF" (встроенная опция в Chrome, Safari, Firefox)

### Что будет в PDF
- Заголовок "Итоги менторства"
- Дата формирования
- Список вопросов с ответами (только заполненные)

### Технические изменения

**Файл: `src/pages/Index.tsx`**
- Добавить функцию `handlePrintPointB()`, которая:
  - Собирает вопросы и ответы в HTML-шаблон
  - Открывает новое окно с этим HTML
  - Вызывает `window.print()` для печати/сохранения в PDF
- Добавить кнопку "Скачать PDF" после списка вопросов в модалке Точки Б (рядом с существующими элементами, перед закрывающим тегом ModalOverlay)
- Кнопка будет неактивна, если ни один ответ не заполнен

Один файл, никаких новых зависимостей, никаких изменений в базе данных.
