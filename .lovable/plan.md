

## Fix: Аудит вулканов не прокручивается на десктопе

### Проблема
Модальное окно "Аудит 7 вулканов" не прокручивается на десктопе. Контент выходит за границы модалки, но скроллбар не появляется.

### Причина
Классическая проблема CSS flexbox: у элемента с `overflow-y-auto` по умолчанию `min-height: auto`, из-за чего он растягивается под размер содержимого вместо того, чтобы ограничиться доступным пространством и показать скроллбар.

### Решение
Добавить `min-h-0` к прокручиваемому контейнеру внутри `ModalOverlay.tsx`.

### Технические детали

**Файл:** `src/components/ModalOverlay.tsx`

Строка 27 — изменить:
```
overflow-y-auto flex-1 p-6 space-y-6
```
на:
```
overflow-y-auto flex-1 min-h-0 p-6 space-y-6
```

Одно изменение в одном файле. Это стандартное исправление для flexbox-контейнеров с прокруткой.
